# Node.js and Express Practice Projects

This repository contains a collection of small Node.js and Express.js projects and experiments. Each folder represents a different topic, feature, or concept used in backend development.  

## üìÇ Folder Overview

| Folder | Description |
|--------|--------------|
| **express-crud/** | Basic CRUD (Create, Read, Update, Delete) operations using Express.js and MongoDB. |
| **express-error-handling/** | Demonstrates error-handling middleware in Express.js applications. |
| **express-error-handling1/** | A variation of the error-handling example with more complex cases. |
| **express-middleware/** | Examples of how to use custom and built-in middleware in Express.js. |
| **express-mongoose-crud/** | CRUD operations using Express.js and Mongoose with MongoDB. |
| **hello-node/** | A simple Node.js application that prints messages or serves a basic response. |
| **mongoose-student/** | Example project using Mongoose models for a student database. |
| **node-mongo/** | Demonstrates connecting Node.js with MongoDB and performing operations. |
| **node-server/** | A simple HTTP server built with Node.js without using Express. |
| **rest-api/** | A simple REST API built with Express.js and MongoDB. |
